Tools

もろもろ必要になりそうなツールをインストール。

1
2
3

$ sudo apt-get install chkconfig
$ sudo apt-get install mpg123
$ sudo apt-get install ethtool

MacからRasPiへファイルのコピー

scpを使います。

1

mac $ scp filename pi@192.168.1.5:~/

WiFi設定

USBのwifiトングルを刺す。

認識されたか確認

1
2
3
4

$ lsusb
Bus 001 Device 007: ID 0411:01a2 BUFFALO INC. (formerly MelCo., Inc.) WLI-UC-GNM Wireless LAN Adapter [Ralink RT8070]
$ ifconfig
wlan0     Link encap:Ethernet .......

こことか参考に無線lanにつなぎます。

$ wpa_passphrase SSID PASS

で出た内容を

/etc/wpa_supplicant/wpa_supplicant.conf の最後にコピペして

さらに

key_mgmt=WPA-PSK

proto=WPA

pairwise=CCMP

group=CCMP

を{}内に追加

最終的にはこんな感じに。。。

$ sudo cat /etc/wpa_supplicant/wpa_supplicant.conf 
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
	ssid="SSID"
	key_mgmt=WPA-PSK
	proto=WPA
	pairwise=CCMP
	group=CCMP
	#psk="PASS"
	psk=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
}

$ sudo cat /etc/network/interfaces 
auto lo

iface lo inet loopback
iface eth0 inet dhcp

allow-hotplug wlan0
iface wlan0 inet dhcp
wpa-conf /etc/wpa_supplicant/wpa_supplicant.conf
iface default inet dhcp

テスト

1
2

$ sudo ifdown wlan0
$ sudo ifup wlan0

んで

1

bound to 192.168.1.32 -- renewal in 3446152 seconds.

とか出ればOK

リブートしても動くかチェックしとくといいかも。

1

$ sudo reboot

https://github.com/Hexxeh/rpi-update
