日経のRaspberry Piセット本が増刷されてたので、買ってみました。

OSのインストール

http://www.raspberrypi.org/downloadsから、Raspbian “wheezy”のイメージをＤＬしました。

一見ＤＬできたかに見えても結構失敗しているようで、SHAのチェックサムは必ずチェックしたほうがいいかと思われます。

ＤＬしたファイルを展開後Win32 Disk Imagerを使って、ＳＤカードに書き込みます。

起動

ＳＤカードとＬＡＮケーブルをさしてから、電源のＵＳＢケーブルを差し込むと起動します。

ＯＳイメージがちゃんと作れてないと、ACTが一瞬点滅してPWRが点灯するもそのままうんともすんともいわなくなるようです。

モニタへの出力も何もありません。

ＤＬしたイメージの確認をしっかりしましょう。

ワタクシ、ノートPCしか手元に無いので、USBキーボード・USBマウスがありません。

ただRaspberry PiのdefaultではDHCPでIPアドレスを取得して、SSHが起動するようなので、

Puttyとか適当なSSHクライアントで接続します。

ちなみにDHCPで割り当てられたアドレスはルータのログで確認しました。

そうするとこんな感じでログインできます。

Please run 'sudo raspi-config'</p> <p>pi@raspberrypi ~ $</p> <p>

“sudo raspi-config”しろとあるので実行。

そうするとこんな感じのメニューが出ます。

じみですな。でもlinuxが走ってるって感じがしますねー。

予想しやすいのを実行してみます。

ファームとか更新

最新のイメージをＤＬしてもアップデートが出ているようです。

$sudo rpi-update

$sudo apt-get update && sudo apt-get upgrade

結構時間がかかります。やっといたほうがいいみたいです。
