Sublime Text 2インストール(Windows)

  • Mar 13, 2014
  • POSTS
B! LINE

Pythonの開発環境ってよく知らない。

いい機会なので、なんか最近話題のSublime Text 2を使ってみます。

無料ではないので良さげだったらお金払います。

ダウンロード

本家に行って、

Downloadページの[Windows 64 bit]を選んでインストーラをダウンロードします。

今日の時点では「Sublime Text 2.0.2 x64 Setup.exe」がダウンロードされました。

3はまだbetaとかなんとか書いてあるところを見ると、活発に開発が進んでるご様子。

インストール

Sublime Text 2.0.2 x64 Setup.exeを実行。

“Add to explorer context menu”はONにしといたほうが良いかも。

後はそのままで。

「C:\Program Files\Sublime Text 2」にインストールされたようです。

実行

特にショートカット等は作られないみたいなので、

C:\Program Files\Sublime Text 2\sublime_text.exeを実行します。

でました。
