Visual Studio Code良いですよね。Sublime(買って)使ってましたが乗り換えようかと思っとります。

随時更新でちょっとしたメモを残しときます。

ユーザー設定

各種設定はユーザー毎の設定ファイル(json)を編集→保存すると即座に反映されるようです。

設定ファイルは

Ctrl+Shift+P user

で開きます。またはメニューのファイル->基本設定->ユーザー設定。

即入れるExtention

Visual Studio Code Settings Sync

1

> ext install code-settings-sync

Auto DownloadとAuto-Upload on changeをOnにしとく

証明書関連のエラー

プロキシとかコンテンツフィルタとか、いろんな事情で証明書関係のエラーが出る。↓こんなの

  • unable to get local issuer certificate
  • Error Logged In Console (Help menu > Toggle Developer Tools). You may open an issue using ‘Sync : Open Issue’ from advance setting command.

ユーザー設定で

1

"http.proxyStrictSSL": false,

それでもだめなら、いったん終了して、コマンドラインから

1
2

set NODE_TLS_REJECT_UNAUTHORIZED=0
