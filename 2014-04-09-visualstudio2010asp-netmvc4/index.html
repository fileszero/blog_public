Visual Studio 2010 + ASP.NET MVC 4

  • Apr 9, 2014
  • POSTS
B! LINE

なんかスマホ系のサイト(つーかRESTfulなサイト?)作るにはMVC4が良いみたいです。なにApiControllerって良さそー。

ただし我がVisual Studio 2010(SP1) 、つるしの状態だとMVC3までのサポートなので、4を使えるようにします。

http://www.microsoft.com/ja-jp/download/details.aspx?id=30683から

AspNetMVC4Setup.exe と AspNetMVC4Setup_JPN.exeをダウンロード。

AspNetMVC4Setup.exe

AspNetMVC4Setup_JPN.exe

の順で実行してみました。

「Processing: Microsoft Visual Studio 2010」のところでやけに時間がかかるので、

タスクマネージャを見ると[VS10SP1-KB2581019-x86.exe]がハングしているような予感（勘です）

インストーラをキャンセルしてPCリブート。

再実行してもダメなので

SP1を再適用…しようとするとディスク容量が不足とか出ます。TEMPをRAM Disk(500M)に割り当ててるからでしょう、たぶん。

そこでコマンドプロンプトを立ち上げ、

D:>cd D:\DownloadFiles\VisualStudio

D:\DownloadFiles\VisualStudio > set TEMP=D:\TEMP

D:\DownloadFiles\VisualStudio > set TMP=D:\TEMP

D:\DownloadFiles\VisualStudio > AspNetMVC4Setup.exe

D:\DownloadFiles\VisualStudio > AspNetMVC4Setup_JPN.exe

で実行したところなんとか終了。所要時間２０分ぐらいでした。

でました
