pip をwindowsにインストール

  • Mar 19, 2014
  • POSTS
B! LINE

pythonにもパッケージ管理ツールがあるみたいで、pipってのが有名なようです。

こちらを参考にインストールします。

get-pip.pyをD:\DownloadFiles\python\にダウンロード

D:\DownloadFiles\python\で作業します。

1
2
3
4
5
6

d:\DownloadFiles\python> python get-pip.py
Downloading/unpacking pip
Downloading/unpacking setuptools
Installing collected packages: pip, setuptools
Successfully installed pip setuptools
Cleaning up...

でpythonのインストールフォルダにpip(とsetuptools)がインストールされるようです。

うちの環境では

c:\Python27\Scripts\pip.exe

です。PATHを通すか悩むところ。。。
