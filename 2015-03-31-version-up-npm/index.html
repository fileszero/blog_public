Mac各種パッケージのバージョンアップ

  • Mar 31, 2015
  • POSTS
B! LINE

とってもべんりなパッケージ管理ソフト、パッケージ管理ソフトも複数あったりして、アップデート方法忘れがち。

まとめときます。

node.js npm

node.jsのパッケージ管理ソフト

そんなnpm自身をアップデートするには

If you update npm itself, do NOT use the npm update command.

The upstream-recommended way to update npm is:

npm install -g npm@latest

だそうです。

やってみる

1
2
3
4
5

Mac ~$ npm --version
1.4.28
Mac ~$ npm install -g npm@latest
Mac ~$ npm --version
2.7.4

上がった。

ちなみに、nodeは

1

Mac ~$ brew upgrade node

または

1
2
3

Mac ~$ nvm ls-remote
#バージョンかくにんして
Mac ~$ nvm install v0.12.1

いっつも忘れるのでメモ。

追記：

1

Mac ~$ npm upgrade -g

で、インストール済みのパッケージをもろもろアップデート。

ruby gem

1

Mac ~$ gem update

HomeBrew

1

Mac ~$ brew update

Vagrant

Vagrant自身はインストーラでアップグレード。

プラグインは

1

Mac ~$ vagrant plugin update
