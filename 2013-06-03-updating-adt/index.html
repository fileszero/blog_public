ADTのアップデート

  • Jun 3, 2013
  • POSTS
B! LINE

前回 git 使ってみる流れでTiny Tiny RSS – Android Clientをビルドしようとしたけど、

ADTアップデートしろと出ました。

こんなん

いいようにコンピュータに使われてますな。

EclipseのHelp-Install new softwareでInstallerを起動。

Work withに「Android Developer Tools Update Site – http://dl-ssl.google.com/android/eclipse/」を選択。

「Developer tools」を選んでNext->Next->Agree->Finish!

アップデートが始まります。

なんかgoogle I/Oで新しい開発ツールの紹介があったもよう。Eclipseとは別物みたいですけど、どんなんでしょか。

とか書いてたら、「No repository found containing: 」とか出てインストールエラー

そーゆーときはレポジトリをアップデート。Window->Preferencesで Install/Update->Available software Sitesで登録されてるサイトを一通りreload

で、再度インストールをば・・・・だめでした。

しょうがないんで、

http://developer.android.com/intl/ja/sdk/installing/installing-adt.html からプラグインを直でダウンロード

Work withのとこのAdd->ArchiveでZIPを指定してみます。

これで一応アップデートできたみたいです。Firewall関係かな～。

ビルドは次回。
