読みやすくていいと思います。プルリク体験できるのもいいっすね。

練習もかねてhtmlファイルを登録してみました。

せっかくなんで公開してみようかと思ったんですが、

素直にファイルのアドレスを開いてもソースが表示されます。

これをwebページとして公開するにはProject Pageを作ります。

GitHubのドキュメントではorphanブランチ作れよってことになってますが、

普通にブランチでもいいみたいです。

1

$ git checkout -b gh-pages
$ echo "public branch" > index.html
$ git add .
$ git commit -am "create public pages"
$ git push origin gh-pages

http://username.github.io/repository/

でページとして表示されます。

cloneすると

他のPCで git clone して git branch しても gh-pages　が出ないんでビビったんですが、

普通に git checkout gh-pages　でチェックアウトできます。
