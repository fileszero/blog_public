Laravel を使ってみる（Windowsセットアップ編)

  • Dec 4, 2015
  • POSTS
B! LINE

ちょっとPHPを触ることになるかもしれないので、お勉強がてら最近のフレームワーク事情を見てたらLaravelってのが人気らしい。

初耳ー。

てことで使ってみます。

諸事情あってWindowsとMac両方にインストールします。まずはWindowsから。

XAMPP

Install XAMPP.

5.6.14 / PHP 5.6.14

Apacheが立ち上がらないときは

こことか見てみると良いかも。

Composer

Install ComposerのInstallerを使用。

インストーラーにもいわれますが、開いてるコマンドプロンプトは閉じる。XAMPP Control Panelも要再起動です。

1
2

# composer -V
Composer version 1.0-dev (feefd51565bb8ead38e355b9e501685b5254d0d5) 2015-12-03 16:17:58

Laravel

こちらを参考にインストール。

作業はXAMMPのShellから。

1

# composer global require "laravel/installer"

%HOME%\AppData\Roaming\Composer\vendor\bin　にPATHを通す。(これは勝手にやってくれてなかった)

1
2

# laravel -v
Laravel Installer version 1.2.2

適当なソースフォルダでlaravel new blogとかやると、blogフォルダが作られて、その下にアプリができます。

なんか出た。

次回Macにセットアップしてみます。
