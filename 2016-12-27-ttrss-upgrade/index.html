Tiny Tiny RSSのバージョンアップ

  • Dec 27, 2016
  • POSTS
B! LINE

RSSリーダーにはTiny Tiny RSSを使っております。

インストールしっぱなしで全然アップデートしておりませんでした。

インストール記事が２０１３年なんで３年放置。。。

そろそろバージョンアップしようと思います。

基本的に最新ソースを持ってきて、config.phpをマージして終わりです。

adminでアクセスするとDBも更新してくれます。

#ワークフォルダに移動
$ cd ~/temp
#DBバックアップ。DB名は環境に合わせてください
~/temp$ mysqldump -u root ttrss > ttrss`date "+%Y%m%d_%s"`.sql
#旧バージョンのバックアップと新バージョンのダウンロード
~/temp$ mv ~/path/to/ttrss ~/path/to/ttrss_v1
~/temp$ git clone https://tt-rss.org/git/tt-rss.git ~/path/to/ttrss
#config.phpはvimdiffでマージしました。
~/temp$ cp ~/path/to/ttrss_v1/config.php ./config.php.old
~/temp$ cp ~/path/to/ttrss/config.php-dist ./config.php.new
~/temp$ vimdiff config.php.old config.php.new
~/temp$ cp config.php.new ~/path/to/ttrss/config.php
#パーミッション設定
~/temp$ chmod -R 777 ~/path/to/ttrss/cache
~/temp$ chmod -R 777 ~/path/to/ttrss/feed-icons/
~/temp$ chmod -R 777 ~/path/to/ttrss/lock/

TT-RSSにadminでログイン

“Database update required” とか出たら更新を実行でDBスキーマが更新されるようです。

かなりモダンなインテーフェースになってるような気がします。
